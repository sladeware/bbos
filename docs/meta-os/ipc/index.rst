####################################
Inter-Process Communication (IPC)
####################################

:Date: |today|

The BBOS mechanism which provides thread communication between BBOS processes
is called Inter-Process Communication or IPC. The IPC service is responsible 
for marshaling the messages to and from local device to remote device, where 
a global device indentifier is used to select between connected devices.
