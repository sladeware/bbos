.. _glossary:

********
Glossary
********

.. if you add new entries, keep the alphabetical sorting!

.. glossary::
   :sorted:

   B3
      Bionic Bunny Build or BBB or B3

   BB
      Bionic Bunny

   BBAPP
      Bionic Bunny Application

   BBOS
      Bionic Bunny Operating System

   BB port
      A message queue for communication between threads.

   BB process
      A particular CPU

   BB thread
      The thread is an atomic unit action within the BB operating system,
      which describes application specific actions wrapped into a single context
      of execution.

   IPC
      Inter-Process Communication

   ITC
      Inter-Thread Communication

   ISR
      Interrupt Service Routine

   TID
      Thread IDentifier
